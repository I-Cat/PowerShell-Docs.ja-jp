# Uninstall-Module

PowerShellGet コマンドレットを使用してインストールされたモジュールをアンインストールします。

## 説明

アンインストール モジュール コマンドレットは、ローカル コンピューターから指定されたモジュールをアンインストールします。 他のいくつかのモジュールに依存している場合、モジュールをアンインストールすることはできません。
アンインストール モジュールのコマンドレットをアンインストールして、モジュールが使用中か場合にも検証します。 モジュールが使用されている場合、エラーがスローされます。

## コマンドレット構文
```powershell
Get-Command -Name Uninstall-Module -Module PowerShellGet -Syntax
```

## コマンドレットのオンライン ヘルプ リファレンス

[アンインストール モジュール](http://go.microsoft.com/fwlink/?LinkId=526864)

### これをオーバーライドすることによって指定 - アンインストール モジュールのコマンドレットで Force オプション

```powershell

```
